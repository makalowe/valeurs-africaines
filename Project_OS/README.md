# Project OS Template

Ce template est la base a copier dans chaque dossier projet:
- Valeurs africaines
- Exam1
- Zion tour
- Papy Peter
- Victor
- Taxi
- tout nouveau dossier

## Setup rapide
1. Copier ce dossier dans le projet cible
2. Renommer le dossier copie en `Project_OS`
3. Completer `01_Vision/PROJECT_BRIEF.md`
4. Completer `02_Roadmap/SPRINT_CURRENT.md`
5. Demarrer la cadence dans `03_Execution/WEEKLY_OS.md`
