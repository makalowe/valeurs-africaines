# Watchlist Sources - Valeurs Africaines

## Regle generale
- Priorite aux sources primaires et institutionnelles.
- Toute information critique doit etre recoupee avec au moins une source independante.

## 1) Institutions africaines
- Union africaine (UA)
- CEDEAO
- CEEAC
- SADC
- IGAD
- Banques centrales/regionales

## 2) Etats (sources gouvernementales)
- Presidence / Primature
- Ministeres des affaires etrangeres
- Ministeres de la defense
- Porte-paroles officiels

## 3) Organisations internationales
- ONU (Conseil de securite, agences)
- Banque mondiale
- FMI
- BAD

## 4) Think tanks et recherche
- IFRI
- ISS Africa
- Chatham House
- CSIS Africa Program
- ECFR

## 5) Medias de reference (recoupement)
- Jeune Afrique
- The Africa Report
- Financial Times (Africa)
- Reuters (Africa)
- AFP

## 6) Verification terrain
- Rapports ONG credibles
- Donnees ouvertes institutionnelles
- Chercheurs de terrain verifies

## Classement interne recommande
- A: source primaire officielle
- B: think tank/recherche credible
- C: media secondaire recoupe
- D: non publiable sans verification supplementaire
