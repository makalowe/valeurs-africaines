# Procédure de publication — Valeurs Africaines

**Workflow obligatoire pour chaque article.** Version du 2026-05-06.

---

## 🔄 Workflow (9 étapes)

```
1. Choix du sujet (utilisateur)
       │
       ▼
2. Cadre editorial obligatoire (Codex)
   ├── Titre propose
   ├── Angle editorial
   └── Structure de l'article
       │
       ▼
3. Validation du cadre par l'utilisateur
       │
       ▼
4. Collecte des sources
   ├── Sources utilisateur
   └── Sources complementaires (Codex)
       └── Priorite : institutions, donnees officielles, think tanks de reference
       │
       ▼
5. Redaction V1
       │
       ▼
6. Corrections utilisateur
       │
       ▼
7. Proposition de 2 illustrations
       │
       ▼
8. Validation illustration
       │
       ▼
9. ❗ Accord explicite de publication
       │
       ▼
10. Depot dans le dossier Drive "valeurs africaines"
```

---

## ⚠️ Regles editoriales

- **Aucune publication finale sans l'etape 9** (accord explicite)
- Sources obligatoires en bas de chaque article
- Double verification minimale avant publication (au moins 2 sources independantes)

---

## 🏷️ Classement des articles

Chaque article doit etre classe selon 2 axes :

### Axe 1 : Pays
Exemples : Mali, Niger, Afrique du Sud, Tchad, Kenya, Maroc, etc.

### Axe 2 : Theme
| Theme | Description |
|---|---|
| Securite | Conflits, defense, peacekeeping |
| Diplomatie | Relations internationales, organisations |
| Renseignement | Cyber, influence, contre-terrorisme |
| Ressources | Petrole, gaz, minerais, terres rares |
| Culture | Langues, identite, souverainete narrative |
| Media | Information, desinformation, communication |
| Lifestyle | Art de vivre, villes, creation |
| Tech | Numerique, innovation, poles tech |
| Economie | Macro-economie, dette, inflation, developpement |
| Politique | Gouvernance, elections, institutions |

## 📄 Conventions de nommage

### Google Doc
```
VA - [Pays] - [Theme] - [Titre court] - AAAA-MM-JJ
```
Exemple : `VA - Mali - Securite - Dossier special Sahel - 2026-05-04`

### Google Sheet
```
VA-SHEET - [Pays] - [Theme] - [Titre court] - AAAA-MM-JJ
```
Exemple : `VA-SHEET - Mali - Securite - Dossier special Sahel - 2026-05-04`

---

## 📋 Checklist de publication

Avant de marquer un article comme "publie" dans la base, verifier :

- [ ] Etape 1 : Sujet choisi
- [ ] Etape 2 : Cadre editorial propose (titre, angle, structure)
- [ ] Etape 3 : Cadre valide par l'utilisateur
- [ ] Etape 4 : Sources collectees (min. 2 independantes)
- [ ] Etape 5 : Redaction V1 terminee
- [ ] Etape 6 : Corrections appliquees
- [ ] Etape 7 : 2 illustrations proposees
- [ ] Etape 8 : Illustration validee
- [ ] Etape 9 : ❗ Accord explicite de publication obtenu
- [ ] Etape 10 : Depose dans le Drive "valeurs africaines"

> **Regle d'or :** Si une seule etape manque, l'article n'est pas publie.
