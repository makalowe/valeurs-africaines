# Prompts IA pour la redaction d'articles

## Template analyse de fond (pour Claude)

```
Tu es Nael T. Koffi, analyste geopolitique specialiste de l'Afrique.
Ecrit un article de 2000-4000 mots sur [SUJET] pour Valeurs Africaines,
magazine geopolitique francophone.

Structure :
1. Accroche (1 paragraphe fort)
2. Contexte et historique
3. Analyse des dynamiques en cours
4. Acteurs et leurs interets
5. Implications regionales et internationales
6. Perspectives

Regles :
- Style : academique mais accessible
- Sources a citer : institutions, think tanks, donnees officielles
- Neutralite journalistique
- Pas de jargon inutile
- Conclusion ouverte (question ou perspective)

Public : diplomes, decisionnaires, chercheurs, etudiants en relations internationales
```

## Template brief actu (pour Claude)

```
Ecrit un brief de 500-800 mots sur [ACTUALITE] pour Valeurs Africaines.

Structure :
- 1 paragraphe de contexte
- Les faits cles (3-4 points)
- Pourquoi c'est important maintenant
- Ce qu'il faut surveiller

Style : concis, factuel, oriente decision.
```

## Template correction/relecture (pour GPT)

```
Relis et ameliore cet article pour Valeurs Africaines :

[COLLER ARTICLE]

Verifie :
1. Clarte du propos
2. Qualite des sources
3. Neutralite editoriale
4. Corrections stylistiques
5. Suggestions d'amelioration
```

## Prompt inspiration sources (pour Perplexity)

```
Trouve les 5 sources les plus recentes et fiables sur [SUJET]
concernant l'Afrique. Donne pour chaque source :
- Titre
- Date
- Organisation emettrice
- Lien
- Resume en 2 phrases

Priorise : rapports institutionnels, donnees officielles,
think tanks de reference (ICG, Chatham House, Brookings, ISS).
```
