# Import RSS - Valeurs Africaines

Fichier OPML:
- docs/va-rss-feeds.opml

## Feedly
1. Ouvrir Feedly
2. Import OPML
3. Selectionner `va-rss-feeds.opml`
4. Verifier les 4 dossiers:
- Securite-Conflits
- Diplomatie-Institutions
- Ressources-Economie
- Renseignement-Influences

## Inoreader
1. Preferences > Import/Export
2. Import OPML
3. Charger `va-rss-feeds.opml`
4. Verifier les categories

## Verification post-import
- Confirmer au moins 1 flux actif par dossier.
- Supprimer/remplacer tout flux qui ne repond pas.
- Ajouter tes sources premium au meme schema.
