# Routine Hebdo Veille - Valeurs Africaines

## Quotidien (lundi-vendredi)

### 07:30-08:00 Scan rapide
- headlines prioritaires
- communiques officiels
- alertes mots-cles

### 12:30-13:00 Tri strategique
- top 3 signaux faibles
- top 2 sujets a fort impact

### 18:00-18:30 Verification
- recoupement faits/chiffres
- qualification des sources (A/B/C/D)

## Hebdomadaire

### Lundi (pilotage)
- definir 1 analyse de fond + 2 briefs
- assigner les angles

### Mercredi (revue qualite)
- verifier robustesse argumentative
- confirmer sources primaires

### Vendredi (publication + retro)
- publier cadence hebdo
- bilan: ce qui a performe / non performe

## KPI de veille
- nb de sujets qualifies / semaine
- taux de sujets publies
- delai moyen collecte -> publication
- taux de corrections post-publication
