# Protocole Veille Internationale - Valeurs Africaines

Objectif: couvrir l'actualite africaine depuis les sources africaines + russes + indiennes + chinoises, sans baisse de rigueur.

## 1) Couverture par blocs linguistiques
- Bloc FR/EN: sources africaines, europeennes, internationales
- Bloc RU: medias/agences russes + think tanks
- Bloc ZH: medias/agences chinoises + institutions
- Bloc IN: medias indiens + analyses geopolitiques

## 2) Règle de lecture
- Toute info non-FR/EN est consideree "signal" tant qu'elle n'est pas recoupee.
- Minimum 2 confirmations independantes avant publication (dont 1 source primaire si possible).

## 3) Pipeline international
1. Collecte multilingue
2. Traduction de travail (note interne)
3. Extraction des faits verifiables (dates, acteurs, citations, chiffres)
4. Recoupement avec sources africaines/institutionnelles
5. Qualification fiabilite (A/B/C/D)
6. Redaction publication-safe

## 4) Matrice fiabilite
- A: communiques officiels / textes institutionnels
- B: think tanks et agences reputees
- C: medias d'opinion ou non recoupes
- D: non publiable sans verification additionnelle

## 5) Règles anti-biais
- Identifier l'agenda probable de la source (interet national, narratif diplomatique)
- Distinguer fait rapporte vs interpretation
- Documenter les contradictions explicites entre blocs de sources

## 6) Format de note obligatoire
- Fait observe
- Source originale (langue + lien)
- Traduction courte
- Niveau de confiance
- Contre-source
- Consequence geopolitique

## 7) Garde-fou publication
- Interdiction de publier une allegation issue d'une seule sphere informationnelle
- Interdiction de publier sans double verification
- Droit de reponse pour les parties directement mises en cause
